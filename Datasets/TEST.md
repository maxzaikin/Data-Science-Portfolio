Обратное распространение (Backpropagation)

После вычисления потерь на каждом шаге, используем обратное распространение для обновления весов WxhWxh​, WhhWhh​, WhyWhy​.
∂Loss∂Why=∑t∂Losst∂yt⋅∂yt∂Why
∂Why​∂Loss​=t∑​∂yt​∂Losst​​⋅∂Why​∂yt​​
∂Loss∂Whh=∑t∂Losst∂ht⋅∂ht∂Whh
∂Whh​∂Loss​=t∑​∂ht​∂Losst​​⋅∂Whh​∂ht​​
∂Loss∂Wxh=∑t∂Losst∂ht⋅∂ht∂Wxh
∂Wxh​∂Loss​=t∑​∂ht​∂Losst​​⋅∂Wxh​∂ht​​
Обновление весов

Весы обновляются с использованием градиентного спуска:
W=W−α∂Loss∂W
W=W−α∂W∂Loss​

где αα - скорость обучения (learning rate).

Этот процесс повторяется для всех временных шагов и всех последовательностей в тренировочном наборе данных. На каждой итерации сеть корректирует свои веса, улучшая предсказания и минимизируя функцию потерь. Таким образом, сеть постепенно учится предсказывать правильные символы на основе входных данных.
